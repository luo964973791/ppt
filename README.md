### islide相关模板
```javascript
PPT快捷键
CTRL +鼠标滚轮放大缩小
CTRL + D    快速复制相同间距的图标
CTRL + M    新建页面
CTRL + G    两元素组合
CTRL + S    保存
CTRL + Z/Y  Z撤销,Y恢复
字体标准间距1.3  行距1.5

SHIFT + 鼠标移动     平行移动
SHIFT + 鼠标拖动     等比例形状
SHIFT + 旋转         每次15°旋转
SHIFT + F5           页面全屏

CTRL + SHIFT + C/V    复制格式

CTRL +[  字体放大
CTRL + ] 字体缩小
CTRL +B 增粗 
CTRL +I 斜体
CTRL +U 下划线
CTRL + E 居中
CTRL + L 居左
CTRL + R 居右


ALT + N + X + H  插入文字
ALT + N + P + D  插入图片
```
